# Contributing

Contributions are welcome:

- typo fixes
- reword suggestions

AI use is OK, but we do not welcome PRs opened by autonomous agents. We'll only review PRs that show clear human intent, from a person who is ready to make themselves available for processing review comments.

## Have an impact on open source

This repo is just the landing page. If you'd like to contribute to the product itself or the libraries it's built on, head over to:

- [Docs](https://github.com/suitenumerique/docs): the product itself
- [Yjs](https://github.com/yjs/yjs): the CRDT engine Docs is built on
- [BlockNote](https://github.com/TypeCellOS/BlockNote): the editor framework Docs is built on
- [DocSpec](https://github.com/docspec/docspec): the conversion library that we use for imports of .docx and .md
